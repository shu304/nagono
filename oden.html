🍢 おでんマニュアル
← トップへ戻る

※おでん出汁を煮込むときは沸騰させないようにする。

🧅 春玉葱のおでん
  1. タッパーに春玉葱1つと浸るくらいのおでん出汁を入れる。
  2. レンジで8~10分。
  3. 深皿に出汁と一緒に盛る。
  4. ブラックペッパーとチャービルを乗せて完成。
🦪 牡蠣のおでん
  1. 牡蠣4つと出汁を鍋で煮込む。
  2. ※生だと危ないのでしっかりと煮込む。
  3. お皿に大根おろし・柚子皮・わかめ・ポン酢を入れる。
  4. しっかり煮込んだ牡蠣を入れて完成。
🥔 里芋

※完成後、前のおでん出汁を入れて提供

  1. タッパーに里芋を3~4個入れる。
  2. ※大きさによって数は変わる。
  3. 出汁を少し入れる。
  4. レンジで5分。
  5. お皿に里芋を入れて三つ葉を上にのせて完成。
🥬 レタス
  1. 120g程度のレタスを出す。
  2. 沸騰したおでん出汁に入れる。
  3. ※10~20秒程度入れたらOK
  4. お皿に盛る。
  5. バター1切れとブラックペッパー少しをのせて完成。
🥦 ブロッコリー
  1. ブロッコリーを100g程度出す。
  2. おでん出汁と一緒に煮込む。
  3. お皿に盛ってブラックペッパーをかけて完成。
🥗 京水菜
  1. 京水菜を70g程度出す。
  2. ※茎と葉をだいたい35gずつ入れれると良い。
  3. 鍋におでん出汁と梅少しを入れて煮込む。
  4. 沸騰してきたら京水菜を入れる。
  5. ※10~20秒程度入れたらOK
  6. お皿に盛って鰹節をかけたら完成。
🌊 刺身わかめ

※完成後、とろろ昆布をのせて提供

  1. 冷凍庫の刺身わかめを解凍する。
  2. おでん出汁と一緒に煮込む。
  3. ※煮込みすぎないように気をつける。
  4. お皿に盛って白ごまを少しかける。
🥟 餃子

※餃子が破れやすいので気を付ける。

  1. 餃子4つとおでん出汁を煮込む。
  2. ※餃子の皮が鍋にくっついて、破れるときがある。
  3. お皿に盛ってネギをのせたら完成。
🍆 焼きナス
  1. ナス1本と出汁を一緒に煮込む。
  2. 長皿に醤油を少し入れる。
  3. ※ナスは火が通ってるか分かりづらいので、気を付ける。
  4. お皿にナスと出汁少しを入れる。
  5. 1口サイズにカットする。
  6. ネギと鰹節をのせて完成。
🐙 明石焼き

※完成後、前のおでん出汁を入れて提供

  1. タッパーに明石焼きを4つ入れる。
  2. ※明石焼き同士がくっつきやすいので気を付ける。
  3. レンジで5分。
  4. お皿に盛って三つ葉をのせて完成。
🍅 トマト
  1. タッパーにトマト1つと浸るくらいのおでん出汁を入れる。
  2. レンジで5~8分。
  3. 深皿に出汁と一緒に盛る。
  4. チャービルを乗せて完成。
🧊 手造り豆腐
  1. 豆腐150g程度を食べやすい大きさに切る。
  2. ※3~4個に切ると良い。
  3. おでん出汁と梅少しと豆腐を一緒に煮込む。
  4. お皿に盛ってネギと鰹節をのせたら完成。
🐷 豚しゃぶ

※完成後、前のおでん出汁を入れてネギをのせて提供

  1. 冷凍庫の豚肉を解凍する。
  2. 深皿に大根おろし・一味・ポン酢を入れる。
  3. 鍋に水を入れて沸騰させる。
  4. 豚肉をいれる。
  5. ※長い時間入れると固くなるので、すぐに出す。
  6. 深皿に4枚入れたら完成。
🐮 牛すじ
  1. おでん出汁と牛すじ1個を煮込む。
  2. お皿に盛ってネギをのせたら完成。
🍖 角煮
  1. 冷凍庫の角煮をレンジで1分解凍する。
  2. 鍋におでん出汁と豆板醤少しを入れる。
  3. 角煮を入れて煮込む。
  4. お皿に盛ってネギをのせたら完成。
🔥 牛ホルモン
  1. 冷凍庫の牛ホルモンをレンジで1分解凍する。
  2. 鍋にニンニク少しキャベツ1つかみ入れる。
  3. 醤油を少量入れて煮込む。
  4. 後から牛ホルモンを入れて少し煮込む。
  5. ※ホルモンは煮込みすぎると固くなるので気を付ける。
  6. お皿に盛ってネギをのせたら完成。
