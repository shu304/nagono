🫛 茶豆
  1. タッパーに茶豆80gと水少しを入れる。
  2. レンジで4分加熱する。
  3. 2つ重ねたお皿に盛り、塩をかけて完成。
🥚 半熟うずらの照り煮
  1. 小さい深皿・受け皿・木のスプーンを準備する。
  2. 深皿にうずら7個を入れる。
  3. 万ネギと照りダレをかけて完成。
🌕 月見じゃこおろし
  1. 大根おろしを器に盛る。
  2. 中央に卵黄をのせる。
  3. じゃこをのせ、ポン酢をかけて完成。
🥩 砂肝のオイル漬け
  1. 小さいタッパーに砂肝100gを入れる。
  2. レンジで1分加熱する。
  3. お皿に盛り、ネギをのせて完成。
🧊 手作り豆腐の薬味冷奴
  1. 有馬山椒少量と鰹節を多めに入れる。
  2. 醤油を入れて混ぜる。
  3. 豆腐150gを盛る（必要なら切る）。
  4. 混ぜた薬味をかける。
  5. ネギとパプリカパウダーをかけて完成。
🔥 京あげ明太炙り
  1. レンジで1分解凍する。
  2. 玉子焼き器で炒める（焦げ注意）。
  3. 2等分してさらに斜めに切る。
  4. 小皿に醤油を入れる。
  5. 長皿に天紙を敷いて盛り付けて完成。
🦑 炙りイカ
  1. レンジで1分解凍する。
  2. 焼き網で炙る。
  3. 小皿にマヨネーズ・醤油・一味を入れる。
  4. 長皿に天紙を敷いて盛り付けて完成。
🧀 クリームチーズの白和え
  1. 豆腐45gとクリームチーズ35gを混ぜる。
  2. 醤油を少量入れて混ぜる。
  3. 柴漬け25gを加えて混ぜる。
  4. 盛り付けてすりごま3gをかける。
  5. 紫漬物をのせ、木のスプーンを添えて完成。
🧀 クリームチーズの西京味噌漬け
  1. 2切れ取り出す。
  2. 2等分し、さらに斜めに切る。
  3. 盛り付けてネギと鰹節をかけて完成。
🥚 出汁巻き玉子
  1. 卵液を準備する。
  2. 玉子焼き器で巻きながら焼く。
  3. 食べやすく切って盛り付けて完成。
🧊 手造り豆腐
  1. 豆乳1kgとにがり15gを入れて混ぜる。
  2. 蓋をしてスイッチON、10分待つ。
  3. 巻きすを敷いた器に出す。
  4. 三つ葉をのせて蓋をして完成。
